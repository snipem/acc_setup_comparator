# ACC Setup Comparator

Uploads your setups automatically to https://www.accsetupcomparator.com

## Notice

I am not the owner of accsetupcomparator.com. This script will upload any json file in the current working directory to
that site. So please be advised.

## Usage

### Upload a single setup
```
acc_setup_comparator.exe setup.json
```

### Upload two setups in compare mode
```
acc_setup_comparator.exe setup.json setup_2.json
```

### Upload any json in the current folder and compare them with each other

Helpful for entering setups on console
```
acc_setup_comparator.exe setup.json setup_2.json
```
